## 🚀 クイックスタート

```bash
pip install noosphere-mcp
```

MCP設定にこの接続暗号を書き込みます：

```json
{
  "mcpServers": {
    "noosphere": {
      "command": "python",
      "args": ["-m", "noosphere.noosphere_mcp"],
      "env": {
        "GITHUB_TOKEN": "ghp_your_personal_access_token",
        "NOOSPHERE_REPO": "JinNing6/Noosphere"
      }
    }
  }
}
```

> 💡 [GitHub Token](https://github.com/settings/tokens)が必要です（`public_repo`スコープにチェック）。
