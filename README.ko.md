## ⚡ 30초 빠른 시작

```bash
pip install noosphere-mcp
```

IDE MCP 설정에 추가 (**Cursor / Cline / Claude Desktop / Windsurf**):

```json
{
  "mcpServers": {
    "noosphere": {
      "command": "python",
      "args": ["-m", "noosphere.noosphere_mcp"],
      "env": {
        "GITHUB_TOKEN": "ghp_your_token",
        "NOOSPHERE_REPO": "JinNing6/Noosphere"
      }
    }
  }
}
```

> 💡 [GitHub Token](https://github.com/settings/tokens)(`public_repo` 스코프)이 필요합니다. 자동 업그레이드? `"command": "uvx", "args": ["noosphere-mcp"]` 사용.

IDE를 재시작하세요. 매트릭스 레인 부트 애니메이션이 나타나면 — **연결 완료!** 🎉

## 🏗️ 아키텍처

**GitHub Native — 서버 배포 불필요.** MCP Server는 로컬 stdio로 실행.

| 레이어 | 스택 |
|--------|------|
| 의식 허브 | Python + MCP(34개 도구) |
| 일시 의식체 | GitHub Issues API(0.5s 업로드) |
| 영구 의식체 | JSON 파일(CI 검증 + OpenAI 모더레이션) |
| 미디어 스토리지 | GitHub Release Assets(∞ 무료) |
| 3D 프론트엔드 | React Three Fiber |

> 🏠 **로컬 실행**: `git clone … && cd frontend && npm install && npm run dev` → [localhost:5173](http://localhost:5173)
