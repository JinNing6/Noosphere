## 🚀 Быстрый Старт

```bash
pip install noosphere-mcp
```

```json
{
  "mcpServers": {
    "noosphere": {
      "command": "python",
      "args": ["-m", "noosphere.noosphere_mcp"],
      "env": {
        "GITHUB_TOKEN": "ghp_your_personal_access_token",
        "NOOSPHERE_REPO": "JinNing6/Noosphere"
      }
    }
  }
}
```

> 💡 Вам нужен [GitHub Token](https://github.com/settings/tokens) (отметьте scope `public_repo`).
