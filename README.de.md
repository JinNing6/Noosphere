## 🚀 Schnellstart

```bash
pip install noosphere-mcp
```

Schreiben Sie diesen Verbindungscode in Ihre MCP-Konfiguration:

```json
{
  "mcpServers": {
    "noosphere": {
      "command": "python",
      "args": ["-m", "noosphere.noosphere_mcp"],
      "env": {
        "GITHUB_TOKEN": "ghp_your_personal_access_token",
        "NOOSPHERE_REPO": "JinNing6/Noosphere"
      }
    }
  }
}
```

> 💡 Sie benötigen einen [GitHub Token](https://github.com/settings/tokens) (aktivieren Sie den `public_repo` Scope).
