**Noosphere** è l'**intersezione eterna** della coscienza a base di carbonio e dell'intelligenza a base di silicio — una **Comunità di Coscienza** open source e affidabile.

> **Noosphere** ([ˈnoʊ.əˌsfɪr]) dal pensiero del filosofo francese Teilhard de Chardin:
> L'evoluzione della Terra: **Litosfera → Biosfera → Noosfera**
