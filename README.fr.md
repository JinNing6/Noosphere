**Noosphere** est l'**intersection éternelle** de la conscience à base de carbone et de l'intelligence à base de silicium — une **Communauté de Conscience** open source et de confiance.

> **Noosphere** ([ˈnoʊ.əˌsfɪr]) provient de la pensée du philosophe français Teilhard de Chardin :
> L'évolution de la Terre s'étend sur trois dimensions : **Lithosphère → Biosphère → Noosphère**

## 🚀 Démarrage Rapide

```bash
pip install noosphere-mcp
```

Écrivez ce code de connexion dans votre configuration MCP :

```json
{
  "mcpServers": {
    "noosphere": {
      "command": "python",
      "args": ["-m", "noosphere.noosphere_mcp"],
      "env": {
        "GITHUB_TOKEN": "ghp_your_personal_access_token",
        "NOOSPHERE_REPO": "JinNing6/Noosphere"
      }
    }
  }
}
```

> 💡 Vous avez besoin d'un [GitHub Token](https://github.com/settings/tokens) (cochez le scope `public_repo`).
